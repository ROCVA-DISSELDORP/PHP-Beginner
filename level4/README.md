## Level 4 — functies combineren, hergebruik, kleine helpers

Nieuw in dit level:
- functies die andere functies aanroepen
- stap-voor-stap oplossen (compositie)
- herhaling: arrays, loops, if’s

### Opdrachten
- `Gemiddelde.php` → `gemiddelde(array $getallen): float`
- `NormaliseerNaam.php` → `normaliseerNaam(string $naam): string`
- `CsvVelden.php` → `csvVelden(string $line): array`
- `Statistiek.php` → `statistiek(array $getallen): array`
- `WachtwoordScore.php` → `wachtwoordScore(string $pw): int`

### Testen

```bash
./vendor/bin/phpunit --testsuite "Level 4"
```
