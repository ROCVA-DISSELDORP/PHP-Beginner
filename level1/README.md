## Level 1 — variabelen, types, strings, return values

Je leert hier:
- variabelen en types (`string`, `int`)
- eenvoudige bewerkingen met strings en getallen
- functies die **iets teruggeven** (`return`)

### Opdrachten
- `Groet.php` → `groet(string $naam): string`
- `Som.php` → `som(int $a, int $b): int`
- `FormatEuro.php` → `formatEuro(int $centen): string`
- `HerhaalRegels.php` → `herhaalRegels(string $woord, int $aantal): string`
- `MaakZin.php` → `maakZin(string $onderwerp, string $werkwoord, string $object): string`
