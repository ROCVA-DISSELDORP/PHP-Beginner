## Level 3 — loops en arrays

Nieuw in dit level:
- `for`/`foreach` loops
- indexed arrays (`[]`)
- herhaling: strings/nummers + if’s in loops

### Opdrachten
- `SomVanArray.php` → `somVanArray(array $getallen): int`
- `FilterEven.php` → `filterEven(array $getallen): array`
- `AantalKlinkers.php` → `aantalKlinkers(string $tekst): int`
- `Tafel.php` → `tafel(int $n): array`
- `Uniek.php` → `uniek(array $waarden): array`
