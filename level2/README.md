## Level 2 — if/else, vergelijkingen, booleans

Nieuw in dit level:
- `if`, `else if`, `else`
- vergelijkingen (`<`, `<=`, `===`, `>=`)
- booleans (`true`/`false`) en logica

### Opdrachten
- `IsVolwassen.php` → `isVolwassen(int $leeftijd): bool`
- `GrootsteVanDrie.php` → `grootsteVanDrie(int $a, int $b, int $c): int`
- `BerekenKorting.php` → `berekenKortingBedrag(int $totaalCents, bool $isLid): int`
- `FizzBuzz.php` → `fizzBuzz(int $n): string`
- `Verzendkosten.php` → `verzendkosten(int $bedragCents): int`
